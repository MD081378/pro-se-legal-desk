Accessibility

Accessibility Statement

Pro Se Legal Desk aims to make public legal-procedure information easier to read, navigate, and use.

Our Approach

We use semantic headings, readable contrast, keyboard-focus styles, descriptive link text, responsive layouts, and alt text for meaningful images.

Ongoing Improvements

Accessibility is an ongoing effort. As the site grows, pages should be reviewed for keyboard navigation, screen-reader structure, color contrast, form labels, and mobile usability.

Report an Issue

If you have trouble using the site, please use the contact page and describe the page, device, browser, and issue you experienced.
